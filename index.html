Sobre Mim

Olá! Sou Daniel Wolter, aqui de Florianópolis, Desde cedo, descobri minha paixão pela tecnologia e em criar experiências de usuário cativantes e interativas. Minhas habilidades abrangem um conjunto diversificado de tecnologias:

Front-End

Front-End - Sou especialista em HTML, CSS e JavaScript, com destaque para frameworks como ReactJS. Minha experiência inclui o uso de Redux, Context API e React Router para criar interfaces funcionais e estilizadas. Além disso, estou familiarizado com práticas de versionamento usando Git e GitHub, bem como testes com RTL e Jest. Minha abordagem inclui o uso de metodologias ágeis como Kanban e Scrum para garantir a entrega eficiente de projetos.

Back-End

Back-End - Domino NodeJS e Express para desenvolvimento de servidores robustos. Também tenho experiência com Docker e ORM (Sequelize) para gerenciar bancos de dados MySQL. Meu conjunto de habilidades inclui a implementação de testes com Mocha, Chai e Sinon, bem como a segurança usando JWT (JSON Web Tokens). Estou familiarizado com arquitetura de software em camadas para criar sistemas escaláveis e de alta qualidade.
